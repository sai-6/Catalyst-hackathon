SCORING LOGIC (EXPLAIN LIKE A PRO)🎯 Core Components1\. Skill Score (LLM-based)Range: 1–5Strict evaluation rulesNo guessing beyond answer2\. Required Level (JD-based)Extracted from job descriptionRange: 1–5Reflects importance of skill3\. Gap Calculationgap = max(required - score, 0)4\. Confidence Score (KEY INTELLIGENCE)confidence = (avg\_score / 5) \* (1 - (avg\_gap / 5))💡 Interpretation:High score + low gap → high confidenceLow score + high gap → low confidence5\.
